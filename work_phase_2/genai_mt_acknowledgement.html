GenAI & MT Acknowledgement

Generative AI was used in the preparation of this assignment.

How was AI used?

Both ChatGPT and Microsoft 365 Copilot were used to develop the User Stories. Answers to the interview questions are taken directly from the AIs' responses to the prompts with a few minor edits (e.g. remove "Answering as Bob") to ensure the text fit with the tone and language of a real-life interview.

ChatGPT was also used to generate code for:

  • The .value-box element
  • The user stories styling
  • The spacing helper classes, and
  • The navigation menu.

Examples of prompts (other than the interview questions) used are:

  • "Please create css for two styles (.question and .response) that look like two people having a conversation."
  • "Create some css and javascript that gives me a menu like this on on my website. The colour should be --primary-color."

How did AI support learning?

Using AI to generate code allowed me to build results quickly, which meant that I was able to experiment with more ideas. I could sample various styles and use my visceral response to inform my design choices. I could then reflect on why certain styles evoked positive responses while others did not, which will help develop my design preferences.

It also empowered me to experiment with code that is far more complex than I would personally write. This is akin to working with a mentor, someone with several years more experience who can introduce me to concepts and techniques beyond my current skill level. Once inputted into VSCode, I could change or eliminate lines of code to understand how it impacted my design.

Critically evaluate AI's effectiveness

ChatGPT proved very effective at delivering functional code with minimal prompting. In some cases, the code it developed was too anticipatory of my needs. For example, ChatGPT included a hover-effect for my value-box which I didn't prompt for and omitted from my code because the effect felt unnecessary for that element. In general, however, it provided what I requested with only minimal adjustments required.

Both ChatGPT and CoPilot were effective at generating content for the User Stories. There were some surprisingly insightful responses (such as the users' preference for a laptop or tablet over a phone due to the screen size) that will be very useful for informing my approach. However the answers quickly became repetitive despite new characters being inputted into the system. This implies that GenAI may be a good tool for early brainstorming but should not be relied upon to provide a nuanced account of the diversity of the users and their browsing preferences.

In terms of creative design, my chosen GenAI platforms will be limited by the quality of my prompt. Being new to website design, there are terms, techniques and concepts that I am not yet aware of, and my inability to use these in my prompts will determine the AIs' limitations. As my webdesign skills grow, so too will my ability to engineer prompts that will generate better code with improved functionality and aesthetics.
